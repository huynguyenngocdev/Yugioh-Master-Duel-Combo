# Yugi-Oh Master Duel Combo Guides

Chào mừng bạn đến với kho lưu trữ các hướng dẫn combo và cách chơi các bộ bài trong **Yu-Gi-Oh! Master Duel** của tôi

## Danh sách các bộ bài (Table of Contents)

Dưới đây là danh sách các bộ bài đã có hướng dẫn:

1. **D/D/D (Different Dimension Demon)**
   - Mỗi line combo tách riêng thành file `.mmd` kèm metadata (starter/required/endboard).
   - Combo files (Mermaid):
     - [01-machinex-1card-gilgamesh.mmd](<D_D_D Combos/01-machinex-1card-gilgamesh.mmd>) — 1-card Machinex → Gilgamesh core.
     - [02-machinex-gate-kepler.mmd](<D_D_D Combos/02-machinex-gate-kepler.mmd>) — Machinex đặt Gate → Kepler consistency line.
     - [03-zero-king-starter.mmd](<D_D_D Combos/03-zero-king-starter.mmd>) — Zero King pop → SS từ Deck.
     - [04-lance-soldier-bridge.mmd](<D_D_D Combos/04-lance-soldier-bridge.mmd>) — Lance Soldier level-fix/self-revive extender.
     - [05-wise-king-solomon-search.mmd](<D_D_D Combos/05-wise-king-solomon-search.mmd>) — Solomon search Zero King/Machinex/toolbox.
     - [06-zeus-ragnarok-extra-pend.mmd](<D_D_D Combos/06-zeus-ragnarok-extra-pend.mmd>) — Zeus Ragnarok +1 Pend Summon/hand negate.
     - [07-arc-crisis-plan.mmd](<D_D_D Combos/07-arc-crisis-plan.mmd>) — Arc Crisis contact-summon plan.

## Hướng dẫn chung

- Các file hướng dẫn sẽ bao gồm:
  - Giải thích các lá bài chủ chốt.
  - Sơ đồ combo (Mermaid charts).
  - Các lưu ý khi đối đầu với Handtrap hoặc setup bàn đấu (Endboard).

## Đóng góp

Nếu bạn muốn thêm hướng dẫn cho các bộ bài khác, hãy tạo một file `.md` mới với tiền tố `Combo_` và cập nhật vào danh sách này.
